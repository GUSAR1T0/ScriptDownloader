# ScriptDownloader

The implementation of downloader of scripts from WEB-pages

## Environment

* Python 3.6
* Django 1.11.7
* requests

## Example of usage

1. Main case:

> python3 manage.py runserver 0.0.0.0:8080

2. Docker Compose:

> docker-compose build
> 
> docker-compose up
